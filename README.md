Personal Portfolio Website

This project is a personal portfolio website created using HTML and CSS.
It presents basic personal information, educational background, media elements, and a contact form in a clean and structured layout.

Project Overview

The website is designed to demonstrate fundamental web development skills, including page structure, styling, and the use of multimedia elements. It follows a simple, readable, and visually appealing design suitable for beginners.

Features

Animated gradient background using CSS

Profile section with a linked image

Education details displayed in a list format

Embedded audio and video elements

Simple query form for user input

Responsive layout for different screen sizes

Technologies Used

HTML5

CSS3

Google Fonts (Poppins)

Project Structure

index.html – Main HTML file

Image.jpeg – Profile image

audio.mp3 – Audio file

captions.vtt – Subtitle file for video

How to Run the Project

Download or clone the project folder

Ensure all required files are in the same directory

Open index.html in a web browser

Notes

The contact form is for design purposes only and does not include backend functionality

Media files can be replaced as required

The project can be extended using JavaScript or backend technologies
